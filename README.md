# Burnout Detector

번아웃 감지 연구를 위한 행동 데이터 수집용 VS Code 확장 프로그램입니다.

## 기능 (Features)
이 확장은 개발자의 행동 패턴을 분석하여 번아웃 징후를 감지하기 위해 다음과 같은 데이터를 익명으로 수집합니다.
1. **타이핑 속도 변화**: 초 단위의 타이핑 빈도 기록
2. **에러 해결 패턴**: 에러 발생 후 해결까지 걸리는 시간 및 에러 유형(단순/복잡) 분석
3. **수정 반복 행동**: Undo/Redo 사용 횟수 측정

## 데이터 수집 및 개인정보 보호
- 모든 데이터는 **익명화된 사용자 ID**와 함께 수집됩니다.
- 수집된 데이터는 오직 **개발자 번아웃 감지 연구 목적**으로만 사용됩니다.
- 확장이 활성화될 때 데이터 수집 동의 여부를 묻는 창이 나타나며, 사용자가 거부할 수 있습니다.

## 사용 방법
1. 확장을 설치합니다.
2. 처음 나타나는 알림창에서 데이터 수집에 동의합니다.
3. 평소처럼 코딩을 진행하시면 데이터가 자동으로 수집되어 서버로 전송됩니다.
